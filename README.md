# Campus Camino A La Universidad - Videojuego Educativo

Proyecto universitario desarrollado en Unity como parte del curso de Ingeniería de Software y Fundamentos Tecnológicos.

---

#  Descripción

Campus Camino A La Universidad es un videojuego educativo en 3D orientado al aprendizaje interactivo mediante misiones, exámenes, puzzles y exploración dentro de un entorno universitario.

El jugador puede recorrer diferentes escenarios, interactuar con NPCs, completar actividades académicas y resolver desafíos relacionados con programación y lógica.

#  Cómo ejecutar el proyecto

1. Clonar el repositorio:

```bash
git clone URL_DEL_REPOSITORIO
```

2. Abrir Unity Hub

3. Agregar el proyecto

4. Abrir la escena principal:

```plaintext
Assets/_Project/Scenes/Juego.unity
```

5. Ejecutar el proyecto desde Unity
